option gui_sample

Default: true
When an object's display is suppressed, when set to to true this will display some sample data in the overflow message. Associated with each object is a size, according to its kind: item command, constraints: length of the text string instance: rows in the algebra colimit schema: size of the schema graph: nodes + edges typeside: types + functions + equations schema: entities + attributes + fks + equations mapping, transform, query: size of source
