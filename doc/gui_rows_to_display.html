option gui_rows_to_display

Default: 32
Sets the maximum number of rows to display per table per instance in the viewer. (I.e., the table in the viewer is conceptually similar to a SQL LIMIT query for the given number of rows). Associated with each object is a size, according to its kind: item command, constraints: length of the text string instance: rows in the algebra colimit schema: size of the schema graph: nodes + edges typeside: types + functions + equations schema: entities + attributes + fks + equations mapping, transform, query: size of source
